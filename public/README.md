# public フォルダ（画像の置き場所）

ここに置いた画像は、そのままの名前でアプリから読み込まれます。

## ロゴ／カードの裏面

- **`card-back.png`** … トップ画面のロゴと、対戦中の「カードの裏面」（相手の手札・山札）に使われます。
  - 例のカード裏面デザイン画像を、名前を `card-back.png` にしてここに入れてください。
  - 縦長のカード形の画像がきれいに収まります。
  - 置かなくてもアプリは動きます（その場合はロゴは文字タイトル、裏面はティール色の模様になります）。

## カードの表面（各発電所の絵）

- `cards/` フォルダに「カードID.jpg」で置きます。詳しくは `cards/README.md` を参照。
